# go_uninstall
GOBINにインストールされている実行ファイルを削除する

## インストール

```
go install github.com/kznagamori/go_uninstall@latest
```

## 使い方

| コマンド       | 概要                                                       |
| -------------- | ---------------------------------------------------------- |
| list           | インストールされているツール（実行ファイル）一覧を表示する |
| 実行ファイル名 | 指定した実行ファイルを削除する                             |
